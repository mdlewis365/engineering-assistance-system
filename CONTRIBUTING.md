# Contributing

This repository accepts documentation improvements only.

## Rules

- Keep content public-safe and conceptual.
- Do not add source code, prompt text, route names, schemas, raw logs, raw uploads, credentials, or provider details.
- Do not publish real engineering cases unless fully authorized and sanitized.
- Do not claim deterministic computation is certified engineering approval.
- Do not describe legacy paths as current primary workflow.
- Preserve human-review and limitation language.

## What Is Not Included

Contributions must not include private computation-plan schemas, MCM internals sufficient to clone the engine, proprietary governance rules, authentication details, model-routing internals, or operational artifacts.
