# Glossary

- EAS: Engineering Assistance System.
- EDR: Engineering Decision Record, a compact decision-relevant summary.
- Engineering Advisory Report: final user-facing EAS report.
- Governance Gate: deterministic classifier for reporting status and human-review requirement.
- MCM: Math Computation Module.
- Run Health: compact summary of computation status, missing data, unit issues, constraints, and readiness.
- Screening: deterministic comparison of candidate options or concepts.
- Selection: deterministic choice of a candidate or configuration when criteria are sufficient.
- Source Basis: explanation of which documents, assumptions, and computed values support the report.

## What Is Not Included

This glossary does not include private schema names, source code identifiers, route names, prompt labels, or hidden implementation fields.
