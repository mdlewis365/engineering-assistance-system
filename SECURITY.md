# Security Policy

## Scope

This is a public documentation repository. It contains no private runtime code, credentials, raw engineering documents, raw logs, or deployment configuration.

## Reporting Issues

Do not disclose private engineering inputs, customer data, credentials, operational logs, prompt text, exploit details, or raw EAS outputs in public issues.

## What Is Not Included

The repository excludes authentication logic, session handling, API keys, model-provider configuration, raw uploaded files, database schemas, job-store payloads, operational logs, route names, function names, and private governance logic.

## Safety Note

EAS output is advisory. Safety-critical, regulated, code-governed, production, structural, electrical, pressure, aviation, public infrastructure, medical, or life-safety decisions require qualified human review.
